Install

  • Extract the downloaded file (.ZIP)
  • Open the extracted. Copy homemodalwindow directory to Prestashop-root-directory/modules via FTP
  • Notice: The directory that you copy: includes a file named homemodalwindow.php and does not have any other homemodalwindow directory.
  • Go to Back Office -> Modules and click install

Thats all

  • To edit module settings click configuration link under Back office ->modules tab over this module block.
